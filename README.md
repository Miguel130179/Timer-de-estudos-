# Time-de-estudos-
Esse projeto é um timer de estudo feito em python criado no mobile.

Esse projeto foi criado devido a uma necessidade minha, e também com a intenção
de praticar sqlite em python.

para rodar em segundo plano no mobile:
vá em simulador de terminal tipo o termux, logo após isso vá até pasta onde contém o arquivo python
e rode: nohup python timer.py &
